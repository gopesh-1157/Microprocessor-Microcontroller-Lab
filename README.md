# 💻 Microprocessor & Microcontroller Lab

A comprehensive collection of **assembly language programs and interfacing experiments** covering **8086, 8051, 8253, and 8255**, designed for academic lab practice and hands-on learning.

---

## 📂 Repository Structure

### 🔹 8086 Programs

| Folder/File | Description |
|------------|------------|
| `8086/Arithmetic Operation/` | Basic arithmetic programs |
| `8086/Array/` | Array manipulation programs |
| `8086/String/` | String operations |
| `8086/ADC/` | Analog to Digital Converter interfacing |
| `8086/DAC/` | Digital to Analog Converter |
| `8086/Steppermotor/` | Stepper motor control programs |

---

### 🔹 MASM Programs

| File | Description |
|------|------------|
| `MASM/l.asm` | Basic MASM setup program |
| `MASM/equal.asm` | Compare two strings |
| `MASM/filewrite.asm` | File handling operations |
| `MASM/own.asm` | Display string program |

---

### 🔹 8051 Programs

| File | Description |
|------|------------|
| `8051/arithmetic/*.asm` | 8-bit & 16-bit arithmetic operations |
| `8051/*` | Core microcontroller programs |

---

### 🔹 8253 Timer Programs

| File | Description |
|------|------------|
| `8253/mode0.asm` | Timer mode 0 |
| `8253/mode2.asm` | Timer mode 2 |
| `8253/mode3.asm` | Timer mode 3 |

---

### 🔹 8255 PPI Programs

| File | Description |
|------|------------|
| `8255/hai.asm` | Basic I/O interfacing |
| `8255/single.asm` | Single port operation |

---

## 🎯 Objectives

- Learn **8086 Microprocessor Architecture**
- Understand **8051 Microcontroller Programming**
- Work with **8253 Timer IC**
- Interface devices using **8255 PPI**
- Practice **low-level programming concepts**
- Gain hands-on experience with **hardware interaction**
